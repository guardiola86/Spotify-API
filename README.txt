Spotify API
===========

Module that integrates with Spotify API. This is what is done so far:
-Admin interface
-Block with random artists list
-Artist page

To configure the module:
-Go to /admin/config/system/spotify_api
-Select number of artists to display in the block
-Enter client id
-Enter client secret
-Place the block "Spotify API Block" wherever you like
-There are two custom permissions: "Administer Spotify API" and "View Spotify API Artist Page"
-To see an artist page, go to /artist/id. Example: /artist/2DlGxzQSjYe5N6G9nkYghR
